# Tic Tac Toe
An [interactive Tic Tac Toe](https://www.theodinproject.com/lessons/node-path-javascript-tic-tac-toe) JavaScript practice project using objects, factory functions and IIFEs. It covers game state management, updating of player names, and visual feedback for tracking winning lines and player scores. 

## Technology used
* HTML
* CSS (Grid, Flexbox)
* JavaScript (Factory Functions, Object Methods, DOM Manipulation, IIFE)

## Features
* IIFEs for tracking tic tac toe grid state and game state
* Factory function for player creation and score manipulation
* Buttons to rename players
* Win and Draw detection
* Reset functionality for additional rounds
